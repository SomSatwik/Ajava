# Petpooja Billing

A small, focused Java billing demo that demonstrates a simple in-memory billing flow.

![Java Badge](https://img.shields.io/badge/language-Java-blue)

## Features
- Simple product model
- In-memory billing example
- Easy to compile and run locally

## Files
- `billing/BillingSystem.java` - core billing logic
- `billing/DB.java` - simple data store
- `billing/Main.java` - launcher / example run
- `billing/Product.java` - product model

## Requirements
- Java 8 or newer

## Build & Run
From the project root (where the `billing` folder lives) run:

```bash
javac billing/*.java
java billing.Main
```

Adjust package/paths if your environment uses packages.

## Example Output
When you run `Main` you should see a simple billing summary printed to the console showing products and totals.
